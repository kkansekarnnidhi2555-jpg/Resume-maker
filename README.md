# Resume Builder Web App

## Features
- Live Resume Preview
- Multiple Templates
- Strength Mode (Basic / Intermediate / Strong)
- PDF Download
- Authentication (PHP + MySQL)
- Save & Edit Resumes

## Tech Stack
- React.js
- Tailwind CSS
- PHP
- MySQL

## Run Project

### Frontend
npm install
npm run start

### Backend
Run using XAMPP / Apache

### Database
Import schema.sql
